# sample.postproc.py

`xxx.postproc.py` is an optional post-processing script for `xxx.txt`.

If `xxx.postproc.py` exists, `llmj-generate.py` automatically loads it and calls the `postproc()` function after each LLM generation.

## Function

```python
def postproc(in_text):
```

### Parameters

| Name | Description |
|------|-------------|
| `in_text` | Text generated by the LLM. |

### Return value

| Return value | Behavior |
|--------------|----------|
| `str` | Accept the text as the final generation result. |
| `None` | Discard the result and retry the LLM generation. Retries are limited by the `--postprocRetry` option of `llmj-generate.py`. |

## Example

```python
def postproc(in_text):
    return in_text
```

The generated text is accepted without modification.

## Retry example

```python
import json

def postproc(in_text):
    try:
        json.loads(in_text)
    except Exception:
        return None
    return in_text
```

If the generated text is not valid JSON, the generation is retried.
